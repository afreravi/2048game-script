How to Play: 2048 is a simple, enjoyable and thoughtful puzzle game. Use your arrow keys or swipe to move the tiles. Get two of the same numbers next to each other and add them together like, 2+2 => 4, 4+4 => 8, 8+8 => 16 and so on. The goal is to get a single tile to be 2048. Isn't it simple!

This game is mobile friendly and compatible with iPhone, iPad or any other smartphone.

NOTE: Game is based on original 2048 game and was created by Gabriele Cirulli. It was released as a free and open-source game
